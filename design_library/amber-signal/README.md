# AMBER SIGNAL - Design System

## Overview
A sophisticated, minimalist design system for professional audio plugins. Features a dark UI with technical monospace typography, subtle depth, and striking neon amber accents, optimized for complex modular interfaces.

## Palette
- **Primary**: #FFC300
- **Secondary**: #00FFF0
- **Background**: #101014
- **Surface**: #1A1A1F

## Components
- Knob Style: futuristic
- Fader Style: cyber
- Border Radius: 4px
